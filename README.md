# TP8 - Banc MCC-MS

## Lancer avec Binder

[![Binder](https://2i2c.mybinder.org/badge_logo.svg)](https://2i2c.mybinder.org/v2/gh/pierrelouis-cmrt/TP8/main?urlpath=%2Fdoc%2Ftree%2FTP8_notebook.ipynb)

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
jupyter lab
```
